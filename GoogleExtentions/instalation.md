# Instalation
### Here is a detailed description how to download extentions and use them step by step
> For Windows 10
1. Find the extention you want to download and open a file
> ![](https://media.discordapp.net/attachments/1163867500623179798/1451742792329068726/image.png?ex=69474855&is=6945f6d5&hm=d73eed84fd08a0311ed68647ebf6333ecbc38bdceb1f2881265c5271da8122b1&=&format=webp&quality=lossless&width=201&height=249)
2. Download each file raw
> ![](https://media.discordapp.net/attachments/1163867500623179798/1451744702960242708/image.png?ex=69474a1d&is=6945f89d&hm=ff1a08a98c516362668238809676a7d27898d75818e1642c11ad0b45f21b0a0c&=&format=webp&quality=lossless&width=1271&height=563)
3. Put them all the downloaded files in 1 folder
4. Open your browser
5. Navigate to extentions
> On chrome *chrome://extensions/*
6. Click on load unpacked
> ![](https://media.discordapp.net/attachments/1163867500623179798/1451745794825912420/image.png?ex=69474b21&is=6945f9a1&hm=1f08754dbcd22115f81cd7c1940bbff28fe864fc2189b815e98f3818aeda9d97&=&format=webp&quality=lossless&width=257&height=285)
7. Select the **created folder** with all the downloaded files
8. Activate extention if not active
> [!NOTE]
> If the file changes something on a webside, **refresh the site to see changes!**
